KiCad Schematic Component Manager User's Manual

General

Schematic Component Manager (SCM) is intended for simplifying of property editing.

Features:
  • Table representation of component list.
  • Automating of component selection.
    • Decicated Selector tool.
    • Using component property as selections criteria.
    • Flexible rules for property matching: same, different, regular expression.
    • Use component as template for property list.
      •
  • Group editing of selected componets.
    • Convenient Inspector tool.
    • Add/Delete/Rename of component[s] properties.
    • Property value substitutions.
    • Property templates: generating property value using template.
      •
  • Customization of components table view.
  • Delicate schematic files handling - preserve file sturcture over read/write operations.
  • Multisheet schematic support.
    •

Description:

Schematic Component Manager contains three tools:

  • Components Table. Represents all schematic components in two-columns table view: Ref | Description.
  • Selector. Offers some capabilities to select components. Selected components are highlighted in Component Table tool, automatically loaded in Inspector Tool (see below) and selected component count displayed in status line.
  • Inspector. Allows group property ediding of selected components. Inspector consists of two subtools: Property Inspector and Field Inspector.
