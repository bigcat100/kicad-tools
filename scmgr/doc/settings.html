Settings Dialog

Components Table View

Componets Table Description column contents can be configured to achieve informative and convenient view. The user can specify template string for each componet reference prefix. Template string can contain plain text and references to other properties of the component. The final description string is constructed from template using text substitutions where references to properties are replaced with the values of the corresponding properties.

For example, some component is capacitor (component reference prefix is C) and has the following properties:

Property Name Property Value
Ceramic Type X7R
Footprint CAP0805
Nominal 0.1uF
Voltage 50V

Template string:

"Cap $Nominal, $Voltage, ${Ceramic Type}, $Footprint"

will produce the final string:

"Cap 0.1uF, 50V, X7R, CAP0805".

Any component type (prefix) can be configured by similar manner.

Components Table Description column settings editor is designed as a table consisting of two columns: components reference prefix and template string. To add item the user should double-click left mouse button or press F2 key on empty cell. To delete the item - press Delete key, this will remove seleted row from the table. Any cell can be edited by double-click left mouse button or pressing F2 key.

Components which reference prefixes are not present in the table will use "LibRef" property value as default.

Component Ignore List

Component Ignore List is simple one column list containig component reference prefixes. Any component that has matching reference will not be loaded in Components Table.

To add item to the list press Insert key. To delete item press Delete key.
